# Gym Tracker

Мобильный трекер тренировок под GitHub Pages. Первая версия хранит данные локально в браузере, показывает цикл Пн/Ср/Пт, дает менять веса и отмечать упражнения как выполненные или пропущенные.

## Команды

```bash
npm install
npm run dev
npm run build
```

## Деплой на GitHub Pages

1. Запушить репозиторий в GitHub.
2. В настройках репозитория открыть Pages.
3. В качестве source выбрать GitHub Actions.
4. После пуша в `main` workflow сам соберет и опубликует `dist`.

## Что уже есть
